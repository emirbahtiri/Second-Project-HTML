Lists

HTML has 3 types of lists:

1. Unordered List
2. Ordered List
3. Description List.


1. Unordered List

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

2. Ordered List

  1. Item one
  2. Item two
  3. Item three
  4. Item four
  5. Item five

3. Description List.
