Images

In HTML we have 3 alternatives how to insert an image:
1. With tag img
2. With tag figure
3. Wtih tag picture



1.Tag img

    This is the basic way to insert an image and this tag can use some attributes like:
  1. src(images link)
  2. alt(if the photo doesn't appear we write a message and mesage will show that text what we wrote)
  3. title (when we go with mouse to image show a title)
  4. width
  5. height


2.Tag figure

This tag has another tag inside (figuration) and it allows us to write a title under the photo.



3.Tag picture

In this tag we can add some images with different size and if we open the webpage in the mobile it will show another image, in laptop another image...
